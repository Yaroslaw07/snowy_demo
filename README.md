## Overview
Snowy Demo is a cozy winter-themed Unity game where players collect coins on a snowy night. Developed with C# and Unity, it showcases fundamental game mechanics in a charming environment.

![image](https://github.com/user-attachments/assets/5981b1c1-88f0-4065-81be-b19a648c2c0e)


## Features ✨
- ❄️ Winter-themed environment with snow effects
- 💰 Simple coin collection mechanics
- 🎮 Basic player controls

## Getting Started 🚀
#### Prerequisites
- Unity 2020.3 or later

#### Installation
- Clone the repository:

```sh
git clone https://github.com/Yaroslaw07/snowy_demo.git
```
- Open the project in Unity.
- Press Play to start the game.
